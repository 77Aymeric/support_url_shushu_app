# Hébergement sur GitHub Pages

Pour héberger ce site gratuitement via GitHub Pages, suivez ces étapes :

1.  Allez sur la page de votre dépôt sur GitHub : [https://github.com/77Aymeric/support_url_shushu_app](https://github.com/77Aymeric/support_url_shushu_app)
2.  Cliquez sur l'onglet **Settings** (Paramètres) en haut.
3.  Dans le menu de gauche, cliquez sur **Pages**.
4.  Sous la section **Build and deployment** > **Source**, sélectionnez **Deploy from a branch**.
5.  Sous **Branch**, sélectionnez **main** et le dossier **/(root)**, puis cliquez sur **Save**.
6.  Attendez quelques minutes. GitHub va vous fournir un lien (généralement `https://77Aymeric.github.io/support_url_shushu_app/`).
7.  Votre site est maintenant en ligne !
